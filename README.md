# Nest JS Starter Kit
This is a starter kit for NestJS. It is a simple project that includes several endpoints and services, making it an excellent starting point for a new project.

## Features
- ENV Support
- Logging Service (File-Based Logging)
- Request Meta Data 
   - Request ID
    - Request IP
    - Request URL
    - Request Method
    - Request TimeStamp
    - Application Version
    - Device Type
    - User Agent etc.

### Installation
```bash
 npm install
```
